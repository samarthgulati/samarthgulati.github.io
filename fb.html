Personalization in Search Engine results

Are users aware of it?

Eli Pariser's TED talk on Filter Bubble, got the interwebs talking on possible cons of personalization in search engine results.

Despite the popularity of the talk, there was no metric to understand whether users are aware of it or how to measure this inherent bias in the results.

Grounded Research

Ran a study on MTurks

Asked the turkers to run given set search queries on Google, Yahoo, Bing, and DuckDuckGo search engines and return the results page's source.

  • Designed a contextual form to collect the data

    Provided instructions based on user's device and browser

    Using the user agent string the page returned provided appropriate instructions for Chrome/Firefox/Safari/IE browsers and MacOS/Windows operating systems.

  • Procedure steps listed upfront

    Color coded for search engine

    The form progress and steps are shown on top with a progress bar which is color coded with the primary color of the search engine.

  • Subtle changes in subsequent steps highlighted

    As most of the steps remained same for subsequent steps except for the link for the query it was easy to miss this subtle change. To bring turker's attention to it, the url animated to get user's attention, telling them it had changed from the previous step.

Visualizing results in Python

Force directed graph showing differences

For a given query and search engine, the variability in results from different turkers is plotted on a force directed graph.
